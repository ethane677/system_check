# software-check

Gather information regarding your computer's resource usage at a regular interval.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Python environment. 
2. Libraries psutil, subprocess, time, sqlite3, and datetime installed if not already.
3. Recomended SQLite database browser of choice. 

Instructions for using the software:

1. Install required databases
2. Run program; possibly resolve errors regarding NVidia cards if necessary.
